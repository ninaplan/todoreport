# Changelog

## [1.0.1] - 예정
### 버그 수정
- 할일 날짜 변경 후 오늘 화면에 잠깐 표시되는 현상 수정
- Notion에서 날짜 변경 시 앱에 반영되지 않는 문제 수정
- Notion에서 날짜 변경된 항목이 기존 날짜 화면에서 새로고침 후 즉시 사라지지 않는 문제 수정
- 로컬 플래너 할일이 SyncQueue에 쌓여 Processor 무한 루프 발생하는 문제 수정

### 성능 개선
- 날짜 이동 시 Notion 재조회 제거 → 로컬 DB만 조회로 즉시 전환
- Notion sync 실행 시점 최소화 (콜드 스타트 / 포그라운드 복귀 / pull-to-refresh / 플래너 전환)
- SyncQueue 무한 루프 제거로 완료 체크 버벅임 개선
