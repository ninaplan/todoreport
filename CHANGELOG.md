# Changelog

## [1.08] - 2026-07-23 (재제출)

### 새 기능
- 투두 날짜 선택을 달력으로 개선 — 날짜별 할일을 카테고리 색 점으로 한눈에 보고, 그날 할일 목록을 바로 확인할 수 있어요
- 달력 카테고리 범례 필터 — 그리드 아래 가로 스크롤(점+이름). 전체·카테고리·미분류 중 하나로 점·목록을 걸러 봐요
- 노션 연동 플래너 달력에서 「불러오기」 — 보고 있는 달의 노션 할일을 월 단위로 가져와요 (아이콘만, 날짜 이동 시 자동 pull 없음)
- 달력·날짜 제목에 첫 안내 말풍선 — 처음 한 번만 보여 사용법을 알려 줘요
- 카테고리 색 팔레트 세트 9종(세트 선택 시 자동 배색, 플래너별 저장)
- 카테고리 색 대비 개선(라이트·다크)

### 버그 수정
- 노션에서 할일을 직접 편집한 뒤 앱 새로고침해도 반영이 안 되던 문제를 개선했어요 (같은 분 편집도 pull에 반영)
- 달력에서 한 달을 불러올 때 동기화 큐가 수백 개로 쌓이던 문제를 막았어요 (범위 pull 시 relation 대량 쓰기 제외)

## [1.0.7] - 2026-07-22 (제출 예정)

### 개선
- 위젯을 작은·중간·큰 크기 모두에서 사용할 수 있도록 통합했습니다 (중간·큰 크기 위젯이 비어 보이면 삭제 후 다시 추가해 주세요)

### 버그 수정
- 할일을 다른 날짜로 옮길 때 간혹 같은 할일이 여러 개 보이던 문제를 수정했습니다
- 노션에서 가져온 할일이 앱에서 사라졌다 나타나며 중복되던 동기화 문제를 개선했습니다
- 위젯에 지난 날짜의 할일이 남아 있던 문제를 수정했습니다

### 기타
- 날짜를 바꿀 때는 앱에 저장된 목록만 바로 보여 주고, 불필요한 노션 재조회를 줄였습니다
- 잠깐 앱을 나갔다 돌아올 때는 노션 동기화를 하지 않고, 5분 이상 백그라운드에 있었을 때만 자동으로 맞춥니다

## [1.0.6] - 2026-07-14 (제출 완료)

### 새 기능
- 플래너 순서 변경·삭제 관리 화면 추가 (설정 → 플래너 → 플래너 관리)

### 버그 수정
- Pro 구독 만료 시 멀티 플래너가 자동으로 다운그레이드되지 않던 문제 수정
- 재구독 시 플래너 잠금이 앱 재시작 없이 즉시 해제되도록 수정
- 노션 연결 해제 시 로컬에 저장된 투두·리포트가 함께 삭제되던 문제 수정 (이제 로컬 데이터는 유지됨)
- 같은 노션 워크스페이스에 여러 플래너를 연결할 때 기존 플래너의 연동이 끊기던 문제 수정

### 기타
- "연동 초기화" 명칭을 "노션 연결 해제"로 변경
- 구독 만료 플래너 선택 안내에서 "읽기 전용"을 "비활성화"로 수정 (선택하지 않은 플래너는 재구독 전까지 사용 불가)

## [1.0.3] - 2026-06-29
### 기능 개선
- 할일/리포트 탭에서 화면 가장자리 스와이프로 날짜·기간 이동 가능 (iOS 기본 뒤로가기와 동일한 방식)

- 리포트 탭 "별점 평균"이 이모지 별 대신 회색 아이콘으로 표시되던 문제 수정
- 별점 표시 시 선택 여부에 따라 별 사이 간격이 미세하게 달라지던 문제 수정
- Notion 연결 초기화 시 일부 인증 정보가 남아있어 재연결이 불안정하던 문제 수정

## [1.0.2] - 2026-06-23
### 버그 수정
- Notion 날짜·시간 양방향 동기화 오류 수정

- 투두 목록 스크롤 시 마지막 항목이 할일 추가(+) 버튼에 가려지는 문제 수정

## [1.0.1] - 2026-06-21
### 버그 수정
- 할일 날짜 변경 후 오늘 화면에 잠깐 표시되는 현상 수정
- Notion에서 날짜 변경 시 앱에 반영되지 않는 문제 수정
- Notion에서 날짜 변경된 항목이 기존 날짜 화면 새로고침 후 즉시 사라지지 않는 문제 수정
- 로컬 플래너 할일이 SyncQueue 무한 루프 발생하는 문제 수정
- 할일 편집 화면에서 긴 제목이 잘리는 문제 수정
- 투두 목록에서 긴 제목 2줄로 제한
- 앱에서 카테고리 생성 시 Notion에 select 옵션이 반영되지 않던 문제 수정
- 카테고리 저장 버튼 중복 클릭 시 카테고리가 두 번 생성되던 문제 수정
- 리포트 탭에서 기간 이동 시 빈 화면이 나오거나 이전 기간 데이터가 남아있던 문제 수정

### 기능 개선
- 날짜 이동 시 Notion 재조회 제거 → 즉시 전환
- isPinned 자동 매핑 alias 확장 (상단고정, 고정, pin, pinned)
- 노션 카테고리 옵션명 변경 시 앱 카테고리 이름 자동 갱신
- 노션에서 삭제된 카테고리 앱에서 보관 처리
- 구독 가격 조정 (월간 ₩6,600 → ₩4,900, 연간 ₩44,000 → ₩33,000)
- 연간 구독 페이월에 "7일 무료체험" 안내 표시 추가
- 페이월 버튼 디자인을 온보딩과 통일 (캡슐 형태)
- 설정 화면 구독 정보 표시 개선 (만료일 표시, 한 줄로 간결화)
